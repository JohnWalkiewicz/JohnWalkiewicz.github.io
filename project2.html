About the Project!

The goal for this project was to use the Google Maps API to locate particular locations around the world. I decided to pinpoint the tallest mountains all arround the world. This was a solo project and took me a little over a week to complete. I located the tallest 20-30 mountains that I found using information I found online. For each mountain, there was a icon in which you can click on. The information presented when you clicked on it was the name, altitude, picture, and a link to more information about that particular mountain. The most difficult part about the project was resizing the mountain icon so that it was visible on the map, but not to large in which it would overlap on other mountains.
